We care much about correctness in the Probabilistic Grammar of Graphics. This is a

With a PGoG specification of the form: (note that these mappings are sorted by the length of their conditionals)

aes[1] <- P(A)
...
aes[n] <- P(X|A, ...)
aes[n+1] <- P(Y|A, ..., X)

Basis

We start from level = 1, or the first line of the specification aes[1] <- P(A).

Suppose the random variable A has levels \(a_{1,1}, ..., a_{1,N}\). Partition in the aes[1] direction, (i.e., width or height) to get \(area_{1,1},..., area_{1,N}\), where \(\forall n \in 1, .., N\), \(area_{1,n} \propto P(A = a_n)\). The resulting visualization is correct.

Inductive Step

level = n

aes[n] <- P(X|A, ...)

Suppose the visualization is correct up to level = n. At level = n, the random variable \(X\) has levels \(x_1, .. x_M\).

Consequently, we have \(area_{n,1},..., area_{1,M}\) partitioned, where \(area_j \propto P(X = x_j)\).
